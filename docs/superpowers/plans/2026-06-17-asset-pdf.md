# 廉政專刊 PDF → 逐項財產申報 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Extract itemized asset-declaration amounts from the 監察院 廉政專刊 gazette PDFs and surface them per legislator, as a human-reviewed pre-fill (assets stay review-gated).

**Architecture:** A new `scraper/lib/gazette.ts` resolves a declaration's 期別 to a public gazette PDF URL, runs `pdftotext` over the relevant pages, and a pure `parseDeclaration(text, name)` extracts category amounts. The asset data model gains an `asset_items` table; `CandidateAsset`/`AssetDeclaration` carry `items: AssetItem[]` instead of a single `total_amount`. The cy adapter populates items; import writes declaration + items; the profile page renders them. Extraction is a pre-fill — assets remain `approved:false` until a human confirms against the PDF.

**Tech Stack:** TypeScript, tsx, `pdftotext` (poppler, already installed), Supabase/Postgres, Vitest. Reuses existing scraper pipeline.

```ts
import type { AssetCategory, AssetItem } from './types';

const CATEGORY_KEYWORDS: Array<[AssetCategory, string[]]> = [
  ['land', ['土地']],
  ['building', ['建物', '房屋']],
  ['deposit', ['存款']],
  ['cash', ['現金']],
  ['securities', ['有價證券', '股票', '基金', '債券']],
  ['investment', ['事業投資', '投資']],
  ['claim', ['債權']],
  ['debt', ['債務']],
];

// Parse a gazette amount like "新臺幣 1,234,567 元" → 1234567; non-numeric → 0.
export function parseGazetteAmount(s: string): number {
  const digits = (s.match(/[\d,]+/g) ?? []).join('').replace(/,/g, '');
  return digits ? Number(digits) : 0;
}

// Slice the text to the named person's block (from their name to the next person/heading),
// then for each category keyword, find the first line mentioning it and read its amount.
export function parseDeclaration(text: string, name: string): AssetItem[] {
  const idx = text.indexOf(name);
  if (idx === -1) return [];
  // block = from the name to the next blank-line-separated section or end (tolerant slice)
  const block = text.slice(idx, idx + 4000);
  const lines = block.split('\n');
  const items: AssetItem[] = [];
  for (const [category, keywords] of CATEGORY_KEYWORDS) {
    const line = lines.find((l) => keywords.some((k) => l.includes(k)) && /[\d,]{2,}/.test(l));
    if (!line) continue;
    const amount = parseGazetteAmount(line);
    if (amount > 0) items.push({ category, amount, label: line.trim().slice(0, 60) });
  }
  return items;
}
```

const PDFTOTEXT = execSync('brew --prefix').toString().trim() + '/bin/pdftotext';

// Download a gazette PDF and return the text of pages [page, page+span].
export async function pdfPageText(url: string, page: number, span = 2): Promise<string> {
  const res = await fetchPolite(url);
  const buf = Buffer.from(await res.arrayBuffer());
  const dir = mkdtempSync(join(tmpdir(), 'gz-'));
  const pdf = join(dir, 'g.pdf');
  const txt = join(dir, 'g.txt');
  try {
    writeFileSync(pdf, buf);
    execFileSync(PDFTOTEXT, ['-f', String(page), '-l', String(page + span), pdf, txt]);
    return readFileSync(txt, 'utf8');
  } finally {
    rmSync(dir, { recursive: true, force: true });
  }
}
